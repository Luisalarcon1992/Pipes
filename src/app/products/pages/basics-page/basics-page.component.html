Todos los pipes hace un cambio visual únicamente, a continuación vemos algunos de los pipes más utilizados.
Para utilizar este pipe se tiene que utilizar la variable:
{{ miVariable | uppercase }} y esto mostrará el texto en mayúscula.

Variable: {{ lowerCase }}: {{ lowerCase | uppercase }}
Para utilizar este pipe se tiene que utilizar la variable:
{{ miVariable | lowercase }} y esto mostrará el texto en minúscula.

Variable: {{ upperCase }}: {{ upperCase | lowercase }}
Para utilizar este pipe se tiene que utilizar la variable:
{{ miVariable | titlecase }} y esto mostrará el texto en capitalizado, es decir, la primer letra de cada palabra estará en mayúscula.

Variable: {{ titleCase }}: {{ titleCase | titlecase }}
